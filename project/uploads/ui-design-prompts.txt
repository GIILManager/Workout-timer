# Workout Timer — UI Design Prompts
# Screen-by-screen descriptions for Claude Code / Lovable / v0
# React Native (Expo) · Dark OLED · Android

=======================================================================
GLOBAL DESIGN LANGUAGE
=======================================================================

App name: Workout Timer
Platform: Android (React Native / Expo)
Mode: Dark only — designed for OLED screens, true blacks
Typography: System font (Roboto on Android). No custom font.
Icon style: Line icons (Lucide or Expo vector icons)

Colour palette:
  bg          #0A0A0A  — screen background (true near-black)
  surface1    #111111  — cards, list rows
  surface2    #1C1C1C  — elevated modals, sheets
  surface3    #262626  — active/pressed states
  green       #22D46E  — SET phase, primary CTA, T1 active ring
  blue        #3B82F6  — BREAK phase, rest ring, secondary CTA
  amber       #F59E0B  — T1 badge, 90-min target warning tint
  red         #EF4444  — overtime ring, 2-hour warning
  text1       #F0F0F0  — headings, big numbers
  text2       #888888  — sublabels, secondary info
  border      #2A2A2A  — card outlines, dividers

Spacing unit: 8px base (8 / 16 / 24 / 32 / 48)
Border radius: cards 12px, buttons 16px, badges 20px
Button height: primary 64px / secondary 48px
Minimum tap target: 44×44px everywhere
Bottom-heavy layout: all primary actions in the bottom third of screen
SafeAreaView: always applied (notch + gesture bar safe)

The app has 5 screens. Each prompt below is self-contained.

SCREEN 2 — ACTIVE WORKOUT (SET phase)
File: app/workout/index.tsx  [phase === 'SET' or 'AMRAP' or 'TIMED']
=======================================================================

Build the main timer screen for when a set is in progress.
This screen must be full-focus with no distractions.
Background: #0A0A0A. SafeAreaView.

--- HEADER (fixed top) ---
Flex row, paddingHorizontal 16, paddingTop 8, alignItems center.

Left: ✕ abandon button — 44×44, icon color text2.
  On press: show confirmation Alert "Abandon workout?" with
  Cancel / Abandon options.

Centre (flex 1): elapsed workout clock.
  Format "0:23:45" — fontSize 17, fontWeight 700, color text1,
  textAlign center, fontVariant tabular-nums.
  If elapsed > 90 min: color amber (#F59E0B).
  If elapsed > 120 min: color red (#EF4444).

Right: compact progress bar.
  Width 80px, height 4px, borderRadius 2, background surface2.
  Filled portion: color green normally, amber near target, red past target.
  Below bar: small label "90m" — text2, fontSize 10.

--- EXERCISE INFO BLOCK ---
paddingHorizontal 24, paddingTop 20, paddingBottom 8.

Top row (flex row, gap 8, alignItems center):
  Exercise name: text1, fontSize 20, fontWeight 700, flex 1, lineHeight 26.
  If TIER1: amber pill badge "T1" same style as home screen.

Bottom row: "SET 2 OF 4  ·  8-10 reps" — text2, fontSize 14,
  fontWeight 500, marginTop 6.

--- CIRCULAR TIMER (centred, fills remaining space) ---
Centred in flex 1 space between info and buttons.
Container: alignItems center, justifyContent center.

SVG circle: size 280×280.
  Background track circle: stroke #1C1C1C, strokeWidth 12, fill none.
  Progress arc: stroke #22D46E (green), strokeWidth 12, fill none,
    strokeLinecap round, strokeDasharray=circumference,
    strokeDashoffset animated (Reanimated useAnimatedProps).
    Arc starts from 12 o'clock (rotate SVG -90deg around centre).
    Fills clockwise from 0 to full as elapsed approaches target.

Centre text overlay (absolute, centred over SVG):
  Main time: formatTime(elapsed) — e.g. "0:47"
    fontSize 52, fontWeight 900, color text1, letterSpacing -2,
    fontVariant tabular-nums.
  Sub-label below: "elapsed" — text2, fontSize 12, fontWeight 500,
    uppercase, letterSpacing 2, marginTop 4.

Target label below SVG: "target: 1:00" — text2, fontSize 12,
  marginTop 12.

--- AMRAP VARIANT ---
When phase is AMRAP:
  Ring shows a 25% arc that rotates slowly (Reanimated withRepeat loop).
  Ring colour: green (#22D46E).
  Centre text: "GO" (until user taps Done, then shows elapsed).
  Target label: "go until failure" in text2.

--- SET DOTS ---
Below the timer + target label, centred, flex row, gap 10.
Each dot: 12px circle.
  Completed sets: filled green circle.
  Current set: filled green circle, subtle pulse scale animation.
  Remaining sets: outline only, border 1.5px, color surface3.

--- BOTTOM BUTTONS (fixed) ---
Position absolute bottom 0, full width.
Background: linear gradient transparent→#0A0A0A over 24px then solid.
Padding: 16 sides, paddingBottom 32 (safe area).

Single DONE button:
  Full width, height 64, borderRadius 16, background green (#22D46E).
  Text "✓  DONE" — color #000, fontSize 17, fontWeight 800.
  Press: scale to 0.97, haptic heavy impact.
  After alert fires (set target reached): button pulses with a subtle
    glow animation (shadow expands and contracts) to indicate it's
    waiting for the tap.

SCREEN 3 — ACTIVE WORKOUT (BREAK phase)
File: app/workout/index.tsx  [phase === 'BREAK']
=======================================================================

Same screen file as Screen 2 but in BREAK phase state.
All layout rules identical. Only these elements change:

--- EXERCISE INFO BLOCK ---
Top row: same exercise name + T1 badge (showing which exercise comes next).
Bottom row: "BREAK  ·  SET 3 NEXT" — color blue (#3B82F6), fontSize 14.

--- CIRCULAR TIMER ---
Progress arc: colour blue (#3B82F6).
Arc depletes clockwise (full at start of break, empty at end).
  strokeDashoffset goes from 0 (full) to circumference (empty).

Centre text: remaining break time counting DOWN.
  e.g. "2:03" — fontSize 52, fontWeight 900, color text1.
Sub-label: "remaining" — text2 label.

--- OVERTIME STATE (when break counter passes 0) ---
Arc colour switches to red (#EF4444). Arc stays at 0 (empty).
Centre text: "+0:45" — color red, fontSize 52, fontWeight 900.
  This is time PAST the break end (counting up again).
Sub-label: "OVERTIME" — color red, fontSize 12, uppercase, letterSpacing 2.

Target label: "target: 2:30" — unchanged, text2.

--- BOTTOM BUTTONS ---
Two buttons stacked, gap 10.

Primary: "▶  START NEXT SET" — full width, height 64, borderRadius 16,
  background blue (#3B82F6).
  Text: color white, fontSize 17, fontWeight 800.

Secondary: "SKIP BREAK" — full width, height 48, borderRadius 12,
  background transparent, border 1.5px solid surface3.
  Text: color text2, fontSize 15, fontWeight 600.

Both buttons stop alert + record actual break duration + advance state.

Preferences screen. SafeAreaView. Background #0A0A0A.

Same pattern as History: back button + "SETTINGS" title.

--- SCROLLVIEW CONTENT ---
Padded 24px sides.

SECTION: "TIMER DEFAULTS"

    Setting name: text1, fontSize 15, fontWeight 500.
    Current value: green (#22D46E), fontSize 15, fontWeight 700.

  Bottom: slider (Slider component), marginTop 12.
    minimumTrackTintColor green, maximumTrackTintColor surface3,
    thumbTintColor green.
    Step: 5 seconds.

Settings in this section (with ranges):
  "T1 Set Duration"    30s – 90s    default 60s
  "T1 Rest Period"     1:30 – 4:00  default 2:30
  "Standard Set"       20s – 60s    default 40s
  "Standard Rest"      1:00 – 2:30  default 1:30

SECTION: "WORKOUT DURATION"
Same section label style.

"Workout Target" row: surface1 card, padding 16.
  Label: text1 fontSize 15.
  Below: segmented control (3 options: "60 min" / "75 min" / "90 min").
    Selected: background green, text black, fontWeight 700.
    Unselected: background surface2, text text2.
    Full width, height 40, borderRadius 8.

"Overtime Warning" row: same card style.
  Segmented: "90 min" / "105 min" / "120 min".

SECTION: "LEARNING"
"Min Sessions to Adapt" row: segmented control "2" / "3" / "5".
Small explanation below: text2, fontSize 12, lineHeight 18.
  "After this many sessions per exercise, the app adapts
   break times to match your actual pace."

"Reset Learning Data" row: surface1 card, padding 16.
  Single pressable full-width row.
  Icon: trash/reset icon, color red.
  Text: "Reset Learned Timings" — color red, fontSize 15, fontWeight 500.
  Sub: "Resets adapted times. Sessions history kept." — text2, fontSize 12.
  On press: Alert "This will reset all adapted timing data. Are you sure?"
